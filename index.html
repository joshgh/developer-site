River Guide

This website was created to serve as a resource for accessing current river information when choosing a place to go whitewater boating. It uses external APIs to access current weather at the river from OpenWeatherMap, current water levels from the US Geological Survey, and directions to the river from Google Maps. This project is built using Drupal 7.

Message Board

This website is a Q&A message board applications. Users are able to post questions to the message board and other users are able to post answers. This project was created as a single page application using Ember.

Coffee Inventory

This application is designed to allow a coffee roastery to track their inventory of beans from receiving shipments of green beans through blending and selling roasted beans. This project was created using Ember.
