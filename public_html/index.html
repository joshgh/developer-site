About

I am a full stack web developer who enjoys working on projects of all sizes.

From simple landing pages to complex line of business applications I can help to make your ideas a reality.

Working to understand a problem domain and incorporating that knowledge into the business logic of an application is my favorite part of the job.

Technologies

I can comfortably work on both the front end and back end of web projects.

The title front end developer can be a little nebulous, my interests and skills skew more towards implementing designs and making things work rather than the actual graphic design aspect.

There are many ways to do most things in web development and I believe that the functionality of an application is more important than the languages and frameworks used to create it. Personal Hobby Projects

River Guide

This website was created to serve as a resource for accessing current river information when choosing a place to go whitewater boating. It uses external APIs to access current weather at the river from OpenWeatherMap, current water levels from the US Geological Survey, and directions to the river from Google Maps. This project is built using Drupal 7.

Message Board

This website is a Q&A message board applications. Users are able to post questions to the message board and other users are able to post answers. This project was created as a single page application using Ember.
